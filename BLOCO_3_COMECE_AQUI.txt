PASSO 1: Setup Stripe
┌───────────────────────────────────────────────────────────────────────┐
│ 1. Criar conta em stripe.com                                          │
│ 2. Gerar API keys (test mode):                                        │
│    - Publishable Key (pk_test_...)                                    │
│    - Secret Key (sk_test_...)                                         │
│ 3. Criar 2 produtos:                                                  │
│    - Pro: R$ 99,00/mês (obter product_id e price_id)                │
│    - Premium: R$ 299,00/mês (obter product_id e price_id)          │
│ 4. Configurar webhook:                                                │
│    - URL: http://localhost:3001/api/payments/webhook                 │
│    - Events: checkout.session.completed, invoice.paid, etc           │
│    - Copiar signing secret (whsec_...)                               │
│                                                                       │
│ ⏱️ Tempo: ~5 min                                                      │
└───────────────────────────────────────────────────────────────────────┘

PASSO 2: Configurar .env
┌───────────────────────────────────────────────────────────────────────┐

│ Adicionar:                                                            │
│ PAYMENT_PROVIDER=stripe                                              │
│ STRIPE_SECRET_KEY=sk_test_...                                        │
│ STRIPE_PUBLIC_KEY=pk_test_...                                        │
│ STRIPE_WEBHOOK_SECRET=whsec_...                                      │
│ STRIPE_PRO_PRODUCT_ID=prod_...                                       │
│ STRIPE_PRO_PRICE_ID=price_...                                        │
│ STRIPE_PREMIUM_PRODUCT_ID=prod_...                                   │
│ STRIPE_PREMIUM_PRICE_ID=price_...                                    │
│ FRONTEND_URL=http://localhost:5173                                   │

❓ DÚVIDAS FREQUENTES
───────────────────────────────────────────────────────────────────────────

P: Como testar pagamento sem cartão real?
R: Use cartão de teste Stripe: 4242 4242 4242 4242
   Expira: Qualquer data futura
   CVC: Qualquer 3 dígitos

P: Onde consigo as API keys do Stripe?
R: dashboard.stripe.com → Developers → API keys

P: Como configurar webhook?
R: dashboard.stripe.com → Developers → Webhooks
   URL: http://localhost:3001/api/payments/webhook
   Events: checkout.session.completed, invoice.paid, etc

P: Como aprovar empresas?
R: Admin acessa /admin/companies
   Seleciona empresa
   Clica "Aprovar"
   Empresa recebe email

P: O que acontece se empresa não tiver is_verified?
R: Vagas são salvas como rascunho
   Aparecem em /admin/companies/pending
   Admin precisa aprovar antes de publicar

P: Como testar middleware de plano?
R: Criar usuário com plan='free'
   Tentar POST /api/ai/analyze-resume
   Deve retornar 403 + mensagem
